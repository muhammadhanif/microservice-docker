# microservice-docker
A simple microservice using Docker container

### Requirements
* Docker Engine
* Docker Compose

### How to run
```
git clone https://github.com/muhammadhanif/microservice-docker

```

```
cd microservice-docker

```

```
docker-compose up

```

### Accessing web app

Open http://your-docker-host-ip-address:8888 in browser.

### Destroy Containers

docker-compose down
